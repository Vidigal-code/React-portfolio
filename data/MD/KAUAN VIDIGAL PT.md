# Kauan Vidigal - Desenvolvedor Full-Stack

### Sobre Mim

Meu nome é Kauan Vidigal, sou um desenvolvedor Full-Stack completo e autodidata, atualmente cursando Ciência da
Computação. Desde 2014, foco em programação com ênfase em Java e Programação Orientada a Objetos.

Como autônomo, ofereço consultoria de TI e desenvolvimento de software e sites personalizados, utilizando PHP com
Laravel para back-end e React ou Vue.js com Vite para front-end. Tenho experiência na configuração de servidores web (
Apache, Nginx) em VPS, segurança avançada (SSL/TLS, firewall), e otimização de performance.

Sou proficiente em bancos de dados MySQL e MariaDB, gerenciamento de DNS, e desenvolvimento de APIs robustas e
escaláveis. Adoto práticas de DevOps para assegurar a eficiência e segurança nos projetos.

Estou aberto a colaborações e desafios que impulsionem meu crescimento profissional e contribuam para o sucesso dos
projetos.

### Experiência Profissional

**Desenvolvedor Full-Stack**

*Período de Atuação:* 2022 - 2024 (Freelancer)

**Desenvolvimento Full-Stack de Sites e Aplicações Web:**

- Utilização de PHP com frameworks como Laravel para o desenvolvimento back-end robusto e escalável.
- Implementação de interfaces dinâmicas e responsivas utilizando React, Vue.js, e Vite no front-end para uma experiência
  de usuário moderna e interativa.
- Integração de APIs e serviços externos para ampliar as funcionalidades das aplicações web.

**Instalação e Configuração de Servidores Web em VPS:**

- Escolha entre Apache HTTP Server ou Nginx para hospedagem de sites e aplicações web.
- Configuração de virtual hosts para suportar múltiplos domínios e subdomínios em um único servidor web.
- Gerenciamento avançado de servidores VPS, incluindo configuração de firewall, SSH e monitoramento de recursos.

**Banco de Dados:**

- Utilização de bancos de dados relacionais como MariaDB e MySQL para armazenamento e gerenciamento eficiente de dados.
- Modelagem de banco de dados, otimização de consultas SQL e implementação de transações para garantir consistência e
  desempenho.

**Segurança e Criptografia:**

- Implementação de SSL/TLS utilizando certificados como Let's Encrypt para garantir comunicações seguras através de
  HTTPS.
- Configuração de políticas de segurança avançadas para proteger dados sensíveis e prevenir ataques cibernéticos.

**Otimização de Performance:**

- Ajustes avançados de configuração para otimização de caching e compressão de recursos estáticos no servidor web.
- Implementação de técnicas de performance como lazy loading, prefetching e pageload optimization para melhorar a
  velocidade de carregamento das páginas.

**Proteção contra DDoS e Segurança Avançada:**

- Configuração de Cloudflare para proteção contra ataques DDoS e mitigação de tráfego malicioso antes de alcançar o
  servidor web.
- Implementação de Proxy Reverso com Cloudflare para melhorar performance, segurança e disponibilidade das aplicações.

**Firewall e Segurança Avançada:**

- Configuração de regras de firewall avançadas no Cloudflare e no servidor para proteger aplicações e dados contra
  ameaças cibernéticas.

**Monitoramento e Análise de Tráfego:**

- Utilização de ferramentas de analytics do Cloudflare para monitorar o tráfego, identificar padrões de acesso e
  detectar possíveis vulnerabilidades.

**Gestão de Hosting e Configuração DNS:**

- Configuração e gerenciamento de serviços de hosting para garantir a disponibilidade e escalabilidade das aplicações
  web.
- Configuração e otimização de registros DNS para direcionar o tráfego corretamente entre os servidores e os serviços
  web.

**Consultoria em TI:**

- Prestação de consultoria para empresas, focando em soluções eficientes e seguras de infraestrutura de TI e
  desenvolvimento de software.
- Implementação de boas práticas de desenvolvimento, DevOps e CI/CD para maximizar a eficiência operacional e garantir a
  qualidade do software.
